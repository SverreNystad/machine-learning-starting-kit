src.ml_service module

This module contains the functions to prepare the data for the machine learning model.

src.ml_service.prepare_data(validation_size: float = 0.1, test_size: float = 0.1, loader: ~src.data.data_loader.DataLoader = <src.data.data_loader.LocalDataLoader object>) → tuple[DataFrame, DataFrame, DataFrame, DataFrame, DataFrame, DataFrame][source]

Prepare the data for the machine learning model. This includes loading the data, feature engineering, and splitting the data into training, validation, and testing sets.

Parameters:

  • validation_size – The size of the validation set. Should be between 0.0 and 1.0.

  • test_size – The size of the test set. Should be between 0.0 and 1.0.

  • loader – An instance of the DataLoader class to load the data.

Returns:

A tuple containing the training, validation, and test data. x_train, x_validate, x_test, y_train, y_validate, y_test.

Example

>>> X_train, X_validate, X_test, y_train, y_validate, y_test = prepare_data()
>>> model.fit(X_train, y_train)
>>> valid_score = model.score(X_validate, y_validate)
>>> test_score = model.score(X_test, y_test)
